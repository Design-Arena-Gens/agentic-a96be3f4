# Agentic Social Share (Web)

This directory contains the Next.js application that powers Agentic Social Share.

## Available Scripts

```bash
npm run dev     # start the local dev server
npm run build   # create a production build
npm run start   # run the production server
npm run lint    # lint the project
```

## Project Highlights

- App Router with TypeScript support
- `/api/generate` endpoint for social copy generation
- Client-side interface with clipboard copy helpers and tone controls

Refer to the root README for detailed documentation and deployment steps.
